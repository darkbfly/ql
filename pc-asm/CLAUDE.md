## 模块概述

## 关键文件

### 图片资源说明

所有 PNG 图片文件用于 PC 端自动化工具的图像识别：

- **数字编号图片**（1.png - 6.png）：可能用于步骤识别或序号识别
- **功能按钮图片**：
  - `close.png` / `close2.png`：关闭按钮识别
  - `finish.png`：完成状态识别
  - `search.png`：搜索功能识别

  - `signed.png`：已签到状态识别
  - `skip.png`：跳过按钮识别

## 使用场景

1. **图像识别**：使用 PyAutoGUI 等工具进行屏幕图像识别
2. **自动化操作**：基于图像识别结果执行点击、输入等操作
3. **状态判断**：通过识别特定图片判断当前页面状态

## 依赖关系

### 使用方式

典型的图像识别代码模式：
```python
import pyautogui

# 查找图片位置
location = pyautogui.locateOnScreen('pc-asm/sign.png', confidence=0.8)
if location:
    # 点击图片中心
    pyautogui.click(pyautogui.center(location))
```

## 配置说明

### 图片要求

1. **格式**：PNG 格式，支持透明背景
2. **分辨率**：建议使用原始分辨率，避免缩放导致识别失败
3. **命名规范**：使用有意义的英文名称，便于代码引用

### 使用注意事项

1. **屏幕分辨率**：图片识别对屏幕分辨率敏感，不同分辨率可能需要不同的图片资源
2. **置信度设置**：`confidence` 参数需要根据实际情况调整（通常 0.8-0.9）
3. **图片更新**：当目标应用界面更新时，需要同步更新对应的图片资源

## 注意事项

1. **版本兼容**：图片资源需要与应用版本保持同步

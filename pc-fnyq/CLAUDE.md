## 关键文件

### `pc-fnyq.py`

**功能**：PC 端自动化主脚本，支持京东、淘宝等平台的自动化操作。

**主要特性**：
- 图像识别和自动化操作
- 验证码处理（拖动验证）
- 暂停/恢复功能（键盘快捷键）
- 时间戳记录

**核心功能**：
1. **图像识别**：使用 `pyautogui` 进行屏幕图像识别
2. **点击操作**：`点击图片中心(path, png, timeout)` 函数
3. **拖动验证**：`淘宝拖动()` 函数处理拖动验证码
4. **暂停控制**：`toggle_pause()` 函数支持暂停/恢复（通过键盘快捷键）

**依赖**：
- `pyautogui`：图像识别和自动化操作
- `keyboard`：键盘事件监听
- `win32gui`, `win32con`：Windows GUI 操作
- `mytool`：工具函数（模块内）

**使用方式**：
```bash

python pc-fnyq.py

# 或使用批处理脚本
run-fnyq.bat
```

## 接口说明

### `点击图片中心(path, png, timeout)`

在屏幕上查找指定图片并点击其中心位置。

**参数**：
- `path`: 图片路径（相对于脚本目录）
- `png`: 图片文件名
- `timeout`: 超时时间（秒）

**返回**：无

### `淘宝拖动()`

处理淘宝的拖动验证码。

**逻辑**：
1. 查找拖动区域图片
2. 执行拖动操作（模拟人工拖动）
3. 处理验证错误情况

### `toggle_pause()`

切换暂停/恢复状态（通过键盘快捷键触发）。

**说明**：使用 `keyboard` 库监听键盘事件，支持暂停和恢复自动化流程。

### `寻找是否存在(path, png, timeout)`

检查屏幕上是否存在指定图片。

**返回**：布尔值，表示图片是否存在
